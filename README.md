\# GenAI Projects



A collection of AI-powered applications built using Python, Streamlit, Gemini API, and Generative AI concepts.



\---



\## Overview



This repository contains multiple Generative AI projects focused on:

\- AI-powered tutoring

\- Image-based AI interaction

\- Conversational AI systems

\- Retrieval-Augmented Generation (RAG)

\- Streamlit-based AI interfaces



\---



\## Projects



\### AI Tutor

An AI-powered tutoring application capable of analyzing uploaded images and providing intelligent explanations using Google Gemini AI.



\### Streamlit Lens

A Streamlit-based application for AI-driven image interaction and analysis.



\### WebRAG

A Retrieval-Augmented Generation project designed for contextual AI responses using external information retrieval.



\### Chatbot

A conversational AI assistant built using Generative AI concepts.



\---



\## Technologies Used



\- Python

\- Streamlit

\- Google Gemini API

\- SQLite

\- PIL (Python Imaging Library)

\- Generative AI

\## Installation



Clone the repository:



```bash

git clone https://github.com/venu-patnaik/GenAI-Projects.git

```



Navigate to the project directory:



```bash

cd GenAI-Projects

```



Install dependencies:



```bash

pip install -r requirements.txt

```



Run a Streamlit application:



```bash

streamlit run ai\_tutor.py

```
